Konsep yang Dipelajari

📱
Viewport Meta Tag
Wajib ada di setiap website modern agar skala layar mobile benar.
<meta name="viewport">
📏
Media Query
Mengubah tampilan berdasarkan lebar layar menggunakan @media.
@media (max-width: 768px)
🗂️
CSS Grid Responsif
Kolom grid berubah: 3 kolom → 2 kolom → 1 kolom sesuai layar.
grid-template-columns
↔️
Flexbox Responsif
Navbar berubah dari horizontal menjadi hamburger menu di mobile.
flex-direction: column
🔡
Fluid Typography
Ukuran teks menyesuaikan layar menggunakan fungsi clamp().
clamp(1.8rem, 4vw, 3rem)
📲
Mobile First Design
Desain dimulai dari layar kecil, diperbesar — best practice industri.
min-width: 769px
