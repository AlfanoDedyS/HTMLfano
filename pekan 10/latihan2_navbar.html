✦ Latihan 2 — Flexbox Navbar

Navbar Dibuat dengan
CSS Flexbox

Navbar di atas menggunakan display: flex untuk menyusun logo, link, dan tombol secara horizontal dan sejajar secara vertikal. Coba perkecil jendela browser untuk melihat tampilan mobile!

🧩 Properti Flexbox yang Digunakan

display: flex
Mengaktifkan Flexbox pada elemen .navbar sehingga child menjadi flex item
justify-content: space-between
Menyebar logo, link, dan tombol ke kiri-tengah-kanan secara otomatis
align-items: center
Meratakan semua item navbar ke tengah secara vertikal
gap: 4px
Memberi jarak antar link navigasi tanpa perlu margin manual
flex-direction: column
Digunakan di mobile agar link tersusun vertikal saat hamburger dibuka
flex: 1
Membuat elemen mengisi sisa ruang yang tersedia (dipakai pada divider)
